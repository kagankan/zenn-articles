# はじめに

Web サイトや Web アプリケーションを作る上で、誰もが使えるようにするためにアクセシビリティを高めることは重要です。
それと同時に、かっこいい・かわいい装飾で目を引くようなデザインにもしたいものです。
アクセシビリティを考慮すると、想定したビジュアルデザインやスタイリングの実現が難しくなることがあり、そのため「アクセシビリティ」と「ビジュアルデザイン」はしばしば対立する要素として語られがちです。
しかし、この課題は技術によって解決できる問題だと考えています。
この本では、アクセシビリティ（特に WCAG AA レベル）を担保しつつ、ビジュアルデザインの選択肢を狭めない実装方法をコンポーネント別に紹介します。

## コード

- コンポーネントごとに HTML, CSS, JavaScript の実装例を示します
- React や Vue 等のフレームワークを使用しない、プレーンな実装を想定します。
- CSS では、 [CSS Nesting](https://developer.mozilla.org/ja/docs/Web/CSS/Guides/Nesting) を使用します。
- JavaScript は TypeScript を使用します
  - ただし、TypeScript の記法は最低限とし、できるだけそのまま JavaScript として動くようにしています。
  - JS で使う場合は、型定義を削除して使用してください
